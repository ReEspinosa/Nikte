# Nikte

Sitio web dedicado a la venta y exhibición de flores. Diseñado con una estética natural y elegante para ofrecer una experiencia de navegación agradable.

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)

## Estructura del proyecto
```
nikte-flores/
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── index.html
```

## Instalación

Clona el repositorio y abre `index.html` en tu navegador. No requiere dependencias ni pasos de build.
```bash
git clone https://github.com/tu-usuario/nikte.git
cd nikte
```

## Uso

Proyecto estático. Puedes desplegarlo en cualquier servicio de hosting que soporte HTML plano, como GitHub Pages, Netlify o Vercel.
